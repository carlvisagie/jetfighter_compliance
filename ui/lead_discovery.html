Lead Discovery Engine

Import → score → review queue → owner-approved outreach only

No automated outreach
Warning

Owner approval required before any human is contacted. This engine does not send email, LinkedIn messages, or spam. It stores and scores candidate leads from lawful manual/CSV sources only.

What it does

Pipeline

1 Import CSV
2 Score & dedupe
3 Review queue (fit ≥ 65)
4 Owner approves
5 Manual outreach + inquiry link

Code: services/acquisition/ · Docs: docs/LEAD_DISCOVERY_ENGINE.md

Run import

Command (local or Render shell)

python scripts/acquisition_import_candidates.py

Input: data/acquisition/leads/import_candidates.csv

Output: data/acquisition/leads/leads.jsonl, leads.csv, review_queue.csv, data/acquisition/reports/latest_discovery_report.md

CSV format

import_candidates.csv columns

company_name,website,contact_name,contact_title,contact_email,linkedin_url,industry,segment,source,source_url,location,notes

segment must be one of:

  • aerospace
  • manufacturing
  • compliance-heavy
  • audit-stressed
  • government-subcontractor
  • quality-ops-manager
Scoring

Rules summary (0–100)

Positive: defense/aerospace/manufacturing, CMMC/AS9100/ISO/ITAR/DFARS/NIST mentions, quality/compliance/ops titles, business email, website.

Negative: consumer industries, spam sources, mega-enterprise names, no contact channel.

Review queue: fit_score ≥ 65 and status new or reviewed. Never auto-approved_for_outreach.

Handoff link: …/ui/inquiry.html?subject=…&ref=<lead_id>

Safe sources only

Allowed

Yes

Owner CSV, public websites, allowed directories, trade pages, manual Sintra lists after review.

No

LinkedIn scraping, login walls, robots.txt violations, bulk spam, automated DMs/email.
