CMMC Level 1 Readiness – Scoring Logic

OVERVIEW This scoring model converts intake answers into a clean 0–100 readiness score. It is intentionally simple and consistent, so you can: • calculate manually • have the backend calculate automatically • insert the score directly into the Readiness Report ------------------------------------------------------------- SCORING VALUES PER CONTROL YES = 3 points PARTIAL = 1 point NO = 0 points UNKNOWN = 0 points ------------------------------------------------------------- 12 KEY CONTROLS (CMMC Level 1 relevant) Access & Accounts 1. Unique user accounts 2. Joiner/mover/leaver process exists 3. MFA enabled on email 4. MFA enabled for remote access Devices & Updates 5. System patching regularly applied 6. Antivirus / EDR installed on all endpoints Backups 7. Backups running 8. Backup restore tested Policies & Documentation 9. Written policies exist 10. Staff acknowledge policies Training 11. Security awareness training performed Vendor Oversight 12. Vendor list / vendor risk process exists ------------------------------------------------------------- MAXIMUM POINTS: 12 controls × 3 points = 36 total points READINESS SCORE FORMULA score = ( total_points / 36 ) × 100 round to nearest whole number ------------------------------------------------------------- COLOR INTERPRETATION GREEN — 80 to 100 Mostly aligned with Level 1 expectations. Minor cleanup may be required. YELLOW — 50 to 79 Partially aligned, but significant gaps exist. Would not pass a validation effort. RED — 0 to 49 Material gaps. Not aligned with CMMC Level 1. Structured remediation required. ------------------------------------------------------------- OPTIONAL ADVANCED WEIGHTING (Use only if you want more realistic scoring in backend automation) Critical items may be weighted higher: • MFA on email: ×1.5 • Patching: ×1.25 • Backups: ×1.25 • Policies: ×1.25 Final formula becomes: weighted_score = (weighted_points / max_weighted_points) × 100 ------------------------------------------------------------- EXAMPLE: Company earns 24 points: 24 / 36 = 0.666 0.666 × 100 = 66 Final Score: 66 (YELLOW)
